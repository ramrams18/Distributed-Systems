Step 1: Run the server.py file in the terminal
Step 2: Run the client.py file in the terminal
Step 3: Then choose the option upload/delete/download/rename
Step 4: Then give the filename for which you want to do the operation
Step 5: If you choose upload, then the file will stored in the uploads folder
        If you choose delete, the file will be deleted from the uploads folder
        If you choose download, the file will be downloaded from the uploads folder to downloads folder
        If you choose rename, the file will be renamed to the new filename in the uploads folder
install requied libraries first such as XMLRPC,PTHLIB and OS
